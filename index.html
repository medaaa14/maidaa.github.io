1. Mengapa menggunakan CSS dan JavaScript dalam pemrograman web?

CSS dan JavaScript digunakan dalam pemrograman web karena masing-masing punya peran penting dalam membuat website menjadi lebih menarik dan interaktif. CSS digunakan untuk mengatur tampilan atau desain halaman web, seperti warna, ukuran font, tata letak, dan lain sebagainya. Tanpa CSS, halaman web hanya akan terlihat polos dengan tampilan default dari HTML. Sementara itu, JavaScript digunakan untuk membuat elemen-elemen di dalam web menjadi lebih dinamis. Misalnya, ketika pengguna mengklik tombol dan muncul animasi, notifikasi, atau terjadi perpindahan halaman secara otomatis tanpa reload, itu semua adalah hasil kerja dari JavaScript.

2. Bagaimana CSS dan JavaScript dapat diintegrasikan dalam pemrograman web?

CSS dapat diintegrasikan langsung dalam HTML melalui tag <style>, atau secara eksternal menggunakan tag <link> ke file .css. Sementara JavaScript bisa disisipkan dalam file HTML menggunakan tag <script>, baik langsung maupun dari file eksternal (.js). Keduanya dapat bekerja bersama untuk menciptakan tampilan dan fungsi halaman yang dinamis.

3. Jelaskan apa saja keuntungan menggunakan JavaScript dan CSS?

Beberapa keuntungan menggunakan JavaScript: Membuat halaman web menjadi interaktif (misalnya: tombol, slider, popup), Meningkatkan pengalaman pengguna (UX), Bisa digunakan untuk validasi form sebelum data dikirim ke server, Bisa digunakan untuk membuat aplikasi SPA (Single Page Application) Keuntungan menggunakan CSS: Memisahkan antara konten dan desain, Membuat tampilan halaman web lebih menarik dan responsif, Mudah digunakan ulang (reusable), Mengurangi duplikasi kode desain

4. Jelaskan apa saja keuntungan menggunakan Bootstrap?

Bootstrap adalah framework CSS yang membantu kita membuat tampilan website yang rapi dan responsif dengan cepat. Beberapa keuntungannya yaitu Menghemat waktu karena sudah disediakan banyak komponen siap pakai, Responsif, otomatis menyesuaikan tampilan di berbagai ukuran layar (PC, tablet, HP), Dukungan grid system yang fleksibel, Tampilan elegan dan modern tanpa perlu ngoding CSS dari nol, Kompatibel dengan semua browser modern
